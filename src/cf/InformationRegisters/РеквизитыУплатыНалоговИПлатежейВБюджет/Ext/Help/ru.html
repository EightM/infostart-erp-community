Банк ® Реквизиты уплаты налогов и иных платежей в бюджет

Регистр предназначен для хранения списка шаблонов для заполнения платежных поручений на уплату налогов и иных платежей в бюджет.

Список шаблонов вводится вручную. При вводе шаблона можно заполнить все данные или только часть данных. Например, предоставляется возможность ввода шаблона по налогу (сбору) для всех организаций. Для этого в шаблоне поле Организация нужно оставить не заполненным.

Данные регистра используются:

  • при вводе платежного поручения на уплату налогов вручную по кнопке Заполнить, доступной после установки флажка Перечисление в бюджет;
  • при автоматическом формировании списка платежных поручений на уплату налогов обработкой Формирование платежных поручений на уплату налогов.

При заполнении платежного поручения анализируется счет учета налогов (сборов), организация и регистрация в ИФНС.
