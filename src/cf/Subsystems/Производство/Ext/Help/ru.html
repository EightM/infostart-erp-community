Учет выпуска по плановым ценам


Если учет производства готовой продукции ведется по плановым ценам с последующим расчетом отклонений от плановых цен, необходимо для всей выпускаемой продукции установить значения плановых цен.
Для этого необходимо:
  1. В справочнике "Типы цен номенклатуры" создать тип цен, который определить как плановый.
  2. Создать документ "Установка цен номенклатуры", в котором определить значения цен для готовой продукции по плановому типу цен.
  3. В документе "Выпуск готовой продукции" настроить типовые операции таким образом, чтобы для проводок по оприходованию готовой продукции бралось значение плановой цены. Плановая цена (если она задана) подставится автоматически при выборе номенклатуры.
